# 💰 대출 이자 계산기

![대출 계산기](https://img.shields.io/badge/version-1.0-blue)

## 소개
주택담보대출, 신용대출의 이자를 쉽게 계산할 수 있는 무료 웹 도구입니다.
복잡한 수식 없이 누구나 쉽게 대출 상환 계획을 세울 수 있습니다.

## ✨ 주요 기능

### 계산 방식
- **원리금균등 상환**: 매월 동일한 금액 상환
- **원금균등 상환**: 매월 동일한 원금 + 감소하는 이자
- **만기일시 상환**: 이자만 납부 후 만기에 원금 일시 상환

### 편의 기능
- 🔢 실시간 계산
- 📊 월별 상환 내역 (처음 12개월)
- 💾 이미지로 저장
- 🔗 링크 공유
- 💬 카카오톡 공유
- 📱 모바일 최적화

## 🚀 사용 방법

1. 대출 금액 입력 (±1억, ±1천만 버튼 활용)
2. 연 이자율 입력
3. 대출 기간 선택
4. 계산하기 버튼 클릭
5. 결과 확인 및 공유
